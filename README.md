## Detaylar
- Veritabanı bağlantılı otel rezervasyon uygulaması.
- Oda tipi ve oda sayısı veritabanında işlenecek şekilde numaralandırıldı.
- Rezervasyon sonrası UUID oluşturulup buna göre kayıt eklendi.
